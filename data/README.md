# Data

The original datasets are not included in this repository because they were provided as part of a technical assessment and may contain proprietary business information.

Expected files:

| File | Description |
|------|-------------|
| `businesscase_customer.csv` | Customer master data |
| `businesscase_account.csv` | Customer account information |
| `businesscase_articles.csv` | Product master data |
| `businesscase_sales_customer.csv` | Transaction-level sales data |
| `businesscase_promotion_customer.csv` | Promotion participation and campaign data |

To reproduce the analysis, place the original CSV files into this folder.
